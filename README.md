The idea behind Delulu is to create a news platform that focuses on positivity and good news.

## Disclaimer

This is a private project and development is slow.

## How to set up

For now, Delulu is offline only (based on Nuxt).

Follow these steps to get it running:

1. Clone the repository
2. Install dependencies
3. Set up environment variables (api keys)
4. Start the development server

### Setup

```bash
# install dependencies
yarn install

# Start the development server
yarn dev
```

### Environment Variables

Copy the `.env.example` file to `.env` and fill in the API keys for the sources you want to use.

## API Sources

The following APIS are currently supported:

- [SerpAPI](https://serpapi.com/)
- [NewsData.io](https://newsdata.io/)

The following APIs are planned for future support:

- [CurrentsAPI](https://api.currentsapi.services/)
- [Animenewsnetwork](https://www.animenewsnetwork.com/encyclopedia/api.php)

## Deployment

Currently there is no deployment setup, but the application can be built for production and previewed locally.

```bash
# Build the application for production
yarn build

# Locally preview production build
yarn preview
```
